# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Development |
| < 0.1   | ❌ Not yet |

## Reporting a Vulnerability

Please report security vulnerabilities to **owiliberyll@gmail.com**

We will respond within 48 hours and coordinate disclosure.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Responsible Disclosure

We believe in responsible disclosure. Please give us 90 days to fix issues before public disclosure.
